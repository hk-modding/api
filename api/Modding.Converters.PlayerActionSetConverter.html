Class PlayerActionSetConverter

JsonConverter to serialize and deserialize classes that derive from PlayerActionSet.
The target class needs to have a parameterless constructor that initializes the player actions that get read/written.
All of the added actions will get processed, so if there are unmappable actions, an IMappablePlayerActions interface should be added to filter the mappable keybinds.
