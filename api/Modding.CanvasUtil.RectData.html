Type Name Description
Vector2 sizeDelta

sizeDelta is size of the difference of the anchors multiplied by screen size so the sizeDelta width is actually = ((anchorMax.x-anchorMin.x)*screenWidth) + sizeDelta.x so assuming a streched horizontally rectTransform on a 1920 screen, this would be ((1-0)*1920)+sizeDelta.x 1920 + sizeDelta.x so if you wanted a 100pixel wide box in the center of the screen you'd do -1820, height as 1920+-1820 = 100 and if you wanted a fullscreen wide box, its just 0 because 1920+0 = 1920 the same applies for height
