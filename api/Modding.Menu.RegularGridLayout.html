Methods

ChangeColumns(int, float, RelVector2?, float)

Changes the column width of the layout, continuing where the layout left off.
This method should generally only be called at the end of a row, because otherwise it may cause an overlap of the currently placed menu items and the new menu items in the same row.
Internally this method resets the actual index count so a copy of index should be saved before calling this method if needed.
